# workstream-vault

Vault documental-operativo para coordinar cambios transversales entre múltiples repositorios.

## Modelo
- System
- Change
- Workstream
- Contract
- ADR

## Regla principal
Las sesiones NO escriben libremente.
Cada sesión solo actualiza su nota `CHG-*.WS-*.md`.

## Carpetas
- `systems/`
- `workstreams/`
- `changes/`
- `contracts/`
- `adrs/`
- `guides/`
- `templates/`
- `logs/`
- `skills/`
- `.atl/`
